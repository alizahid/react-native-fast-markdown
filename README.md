# react-native-fast-markdown

Fast Markdown viewer/editor for React Native

## Installation


```sh
npm install react-native-fast-markdown
```


## Usage


```js
import { FastMarkdownView } from "react-native-fast-markdown";

// ...

<FastMarkdownView color="tomato" />
```
